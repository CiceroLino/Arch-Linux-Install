# Cinnamon no Arch Linux

Cinnamon é ótimo para quem quer um desktop tradicional e simples.

## Instalação

```bash
sudo pacman -S cinnamon lightdm lightdm-gtk-greeter
sudo systemctl enable lightdm
```

## Pacotes úteis

```bash
sudo pacman -S nemo-fileroller xed xreader
```

## Dicas de ricing

- Escolha um tema GTK consistente (claro ou escuro).
- Use ícones Papirus para visual limpo.
- Ajuste o painel para modo compacto.
- Evite applets demais para manter desempenho.

## Ajustes recomendados

- Ative atalho para terminal e gerenciador de arquivos.
- Configure área de trabalho para ocultar ícones se quiser visual minimalista.
